# 项目说明

## 概述

简历模板站是一个专业的在线简历制作平台，为用户提供海量精选简历模板，支持在线编辑和PDF导出功能。

## 开发阶段

本项目将分为多个任务完成：
- Task 1：项目基础结构搭建 ✓
- Task 2：前端页面组件开发
- Task 3：后端API开发
- Task 4-9：功能模块完善
- Task 10：模板设计与完善

## 功能模块

### 1. 首页 (Home)
- 展示平台特色
- 推荐热门模板
- 引导用户注册

### 2. 模板中心 (TemplateList)
- 分类筛选（简约/专业/创意/技术/管理）
- 语言筛选（中文/英文）
- 搜索功能
- 模板预览

### 3. 模板详情 (TemplateDetail)
- 大图预览
- 模板信息展示
- 使用按钮
- 收藏功能

### 4. 简历编辑器 (ResumeBuilder)
- 表单式编辑
- 实时预览
- 保存功能
- PDF导出

### 5. 个人中心 (UserCenter)
- 用户信息
- 简历管理
- 收藏管理
- 会员状态

### 6. 会员中心 (ProPage)
- 会员方案展示
- 价格策略
- 特权说明
- 常见问题

## 数据库设计

### users 表
- 用户基本信息
- 密码加密存储
- 会员状态标记

### templates 表
- 模板元信息
- 分类和语言
- 样式数据（JSON）
- 下载统计

### resumes 表
- 用户简历
- 关联模板
- JSON格式内容

### favorites 表
- 用户收藏
- 联合唯一索引

## 会员体系

- 普通用户：免费模板
- 会员用户：全部模板 + 高级功能
